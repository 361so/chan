# 城市微光 (City Glimmer)

**用镜头记录城市温度，让每一份美好被看见。**

“城市微光”是一个城市文明共建平台，鼓励市民随手拍下社区美景、文明瞬间与公益行动。用微小的力量汇聚成城市文明的璀璨星河。每一次打卡都是对家园的热爱，每一份分享都能点亮城市的文明之光。

**一句话定位**：市民随手拍文明行为、社区美景，完成公益任务，赚积分领荣誉，城市文明共建平台。

## 核心功能

### 1. 文明打卡
用户可拍摄并上报以下三类内容（管理员审核后发放积分）：
- **社区美景** (10分)：绿化、干净街道、小区风景。
- **文明行为** (15分)：礼让、垃圾分类、有序停车。
- **公益行动** (20分)：捡垃圾、劝导、志愿行动。

### 2. 每周任务 (规划中)
后台预设固定任务，系统自动循环发布：
- 拍一处干净整洁的公共环境
- 拍一次文明停车
- 参与一次环保小事
- 发现身边文明行为
(用户完成 → 上传照片 → 管理员审核 → 自动发积分)

### 3. 积分 & 勋章体系
零成本激励体系，积分可用于：
- **升级等级**：提升文明市民等级。
- **兑换勋章**：
    - 文明大使
    - 环保小卫士
    - 美好家园守护者
    - 月度文明之星
- **实物兑换** (后期规划)：对接商家/社区提供的纸巾、洗衣液等小礼品。

### 4. 光荣榜
- **多维度榜单**：日榜 / 周榜 / 月榜。
- **展示面**：全方位展示文明达人风采。

## 运营模式

**“管理员只做【审核】，其它全自动”**

1.  **用户行为**：用户拍照上传。
2.  **审核流程**：管理员在后台（或手机端管理页）点击“通过”或“驳回”。
3.  **自动化处理**：
    - 审核通过 → 系统自动加积分。
    - 内容自动上广场（发现页）。
    - 任务状态自动刷新。
    - 榜单数据自动计算更新。

## 技术栈

- **前端框架**: [UniApp](https://uniapp.dcloud.io/) + [Vue 3](https://v3.vuejs.org/)
- **构建工具**: [Vite](https://vitejs.dev/)
- **状态管理**: [Pinia](https://pinia.vuejs.org/)
- **UI 组件**: [uni-ui](https://uniapp.dcloud.io/component/uniui/uni-ui.html)
- **地图服务**: 腾讯地图 SDK
- **后端服务**: UniCloud (云开发) / 这里的后端取决于具体实现，目前假设为云开发或自建后端

## 目录结构

```
src/
  ├── api/            # API 接口
  ├── pages/          # 页面文件
  │   ├── index/      # 首页 (发现/地图模式)
  │   ├── report/     # 文明打卡页
  │   ├── rank/       # 光荣榜
  │   ├── user/       # 个人中心
  │   └── admin/      # 管理员审核页
  ├── static/         # 静态资源
  ├── store/          # Pinia 状态管理
  ├── utils/          # 工具函数
  ├── App.vue         # 根组件
  ├── main.js         # 入口文件
  └── manifest.json   # 应用配置
```

## 快速开始

1.  **安装依赖**:
    ```bash
    npm install
    ```

2.  **运行开发环境**:
    - H5: `npm run dev:h5`
    - 微信小程序: `npm run dev:mp-weixin`

3.  **配置说明**:
    - 在 `manifest.json` 中配置微信小程序 AppID。
    - 确保腾讯地图 Key 配置正确。
